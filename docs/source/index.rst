========================================
Welcome to the legate-raft documentation
========================================

``legate-raft`` is a prototype of a legate-enabled version of libraft.
The project is still in an exploratory state and mainly contains prototype
versions of naive bayes, knn, and kmeans.

.. note::
   The API layout of this project is considered fully experimental.
